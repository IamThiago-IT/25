🌝 Modo escuro somente CSS🌚

Usando o :checked pseudo-seletor de classe, pode-se definir o estilo de descendentes e irmãos subsequentes de uma caixa de seleção quando ela estiver no estado marcado.

Hter uma caixa de seleção para habilitar o modo escuro e um elemento que contém todos os outros conteúdos da página no mesmo nível da árvore DOM nos permite oferecer um modo escuro somente CSS de uma página da web.

Uma implementação muito boa desta técnica pode ser encontrada em Mode Drak Js. é a base para essa implementação, então todo o crédito vai para ela! 🎉
